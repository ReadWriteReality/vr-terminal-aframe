The HTTPS support is ready. Here's how to use it:               
                                                                              
  1. Restart the server (kill the existing one first):                          
  # Find and kill the running server
  kill $(lsof -t -i:8080) 2>/dev/null                                           
  npm start                                                                   
  2. You should see output like:                                                
  VR terminal server listening on https://0.0.0.0:8080                          
  TLS: enabled                                                                  
  3. From your Apple Vision Pro, open:                                          
  https://10.0.0.67:8080/keyboard_terminal                                      
  3. Accept the self-signed certificate warning when prompted.                
                                                                                
  The server auto-detects the certs in certs/ — if they exist, it uses HTTPS; if
   not, it falls back to HTTP. The WebSocket connection in app.js already adapts
   automatically (wss: for HTTPS, ws: for HTTP).                                
                                                                              
  When you eventually move to GoDaddy, you can either drop the real cert files  
  into certs/ or point to them with environment variables:
  TLS_CERT=/path/to/fullchain.pem TLS_KEY=/path/to/privkey.pem npm start
